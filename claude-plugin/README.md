# Claude Desktop Plugin

Pre-built `.plugin` file for one-click installation in Claude Desktop.

## Install

1. Download `vc2-essentials.plugin` (or get the latest from [Releases](https://github.com/nextwave-partners/vc2-essentials/releases))
2. Open Claude Desktop
3. Go to Settings → Plugins → Install from file
4. Select the `.plugin` file

## Commands

After installation, these slash commands are available:

| Command | What it does |
|---------|-------------|
| `/score-opportunity` | Run the 10-dimension scorecard with autonomous analysis |
| `/generate-safer` | Generate a Safer term sheet with VREC provisions |
| `/simulate-safer` | Run the Safer Scenario Simulator |
| `/simulate-fund` | Run Monte Carlo fund simulation |

## Requirements

- Claude Desktop with plugin support
- macOS or Windows
